## 네비게이션/IA
- 하단 탭(모바일 전용): 홈, 통계, 추천/소셜(옵션), 멤버십(옵션), 내정보. 초기 구현은 홈/통계/설정 3탭으로 단순화 가능.
- 홈 내부 탭: 루틴, 투두. 상단 캘린더 스와이프/탭으로 날짜 전환.
- 상세/편집: 루틴/투두를 탭하거나 메뉴 → 모달 시트/풀스크린 폼으로 수정.

## 화면 설계
- **홈-루틴**: 상단 월/주 달력(가로), 오늘/선택일 강조. 카테고리/태그 칩과 필터. 루틴 리스트는 시간 슬롯 정렬, 체크/스킵/편집 아이콘. FAB(+)로 추가.
- **홈-투두**: 동일 캘린더 + 필터. 투두 리스트는 잠금/반복 아이콘, 완료 체크, 빈 상태 메시지.
- **통계-월간**: 습관 카드 그리드(제목+이모티콘), 달력 히트맵, 완료율/카운트. 월 전환 화살표.
- **통계-주간**: 습관 리스트 + 요일별 체크 히트맵, 하단 합계(달성/쉼/초록불).
- **루틴 편집 모드**: 리스트 아이템 롱프레스 → 인라인 액션(복사/보관/삭제/수정). 스와이프 액션으로도 가능.
- **습관 수정 폼**: 이름 입력, 알림 토글+시간, 반복 규칙, 수행 시각/장소, 태그, 연속달성/트래킹 모드 토글, 공개/비공개, 시간 필터. 저장 버튼은 하단 고정.

## 컴포넌트 패턴
- 캘린더: 주간 슬라이더(오늘 강조), 월 선택 드롭다운.
- 필터 칩: 선택/비선택/잠금 상태, 이모티콘 포함.
- 리스트 아이템: 좌측 체크/잠금, 중앙 제목+세부(시간/메모), 우측 아이콘(알림/메뉴).
- 통계 카드: 제목/이모지 + 소형 달력 뱃지, 하단 완료율 라벨.
- 폼 입력: 섹션 카드 단위로 그룹화, 토글/셀렉트/텍스트필드 혼합, 도움말(?) 아이콘.
- 공용 버튼: 둥근 코너, 코랄 포인트 색상, 버튼 텍스트는 굵게.

## 상태/데이터 흐름
- 전역: 선택된 날짜, 탭 상태(루틴/투두), 필터(태그/기간).
- 루틴/투두: 리스트(당일 전개), 상태(done/skip/defer/lock), 반복 규칙, 알림 시간.
- 통계: 기간별 완료 로그(날짜+상태) 기반으로 월/주 히트맵 계산.
- 편집/추가: 폼 상태 → 저장 시 로컬 스토리지(JSON/SharedPreferences) 반영 + 알림 리스케줄.

## 구축 우선순위
1) 홈 뼈대: 캘린더/탭/리스트/FAB, 더미 데이터 연결.
2) 루틴/투두 항목 컴포넌트 + 체크/편집 액션 훅.
3) 통계 월/주 뷰: 카드/히트맵 기본 레이아웃, 더미 데이터 계산.
4) 습관 추가/수정 폼: 섹션/토글/시간 선택기, 저장 버튼.
5) 상태 관리/스토리지: Riverpod(or Bloc) + 로컬 파일/SharedPreferences 저장, 알림 스케줄링.
6) 빈 상태/가드: 잠금(멤버십) 섹션, 안내 CTA, 로딩/오프라인 처리.
